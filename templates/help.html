How To Use:
- set the ip address and port of the device you want to send commands to (the ion 10.0.0.196 and port 8000)
- set the starting and ending values (range). These will be the cues, macros, ect that will be used for the OSC command
- choose between 'random' or 'in order'. 'Random' will run the numbers randomly where 'in order' will run the numbers in number order (1,2,3,4, ect)
- set the example OSC command. A lowercase 'x' will be the standin for the numbers decided earlier. The command to send OSC commands to the ion to run macros would be "/eos/macro/x/fire"

NOTE: IF YOU DON'T KNOW HOW TO USE OSC YOU SHOULD NOT USE THIS, YOU CAN AND WILL BREAK THINGS
